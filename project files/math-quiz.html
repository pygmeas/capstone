Capstone Summary:

The Simple Math Trainer is an interactive learning tool designed to help young children practice essential arithmetic skills. Built with HTML, CSS, and JavaScript, the app generates random math problems covering addition, subtraction, multiplication, and division. It provides instant feedback, smooth button interactions, and a child friendly layout that keeps the experience engaging. This project highlights effective use of DOM manipulation, event handling, and dynamic content creation, all focused on making math fun and accessible for beginners.
